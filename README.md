# Elon Morsk

Eine schöne Morse-Code App mit echten Tönen und Oszilloskop-Visualisierung.

## Live Demo

[![Deploy Flutter Web](https://github.com/Damagevindi/hm_elon_morsk/actions/workflows/deploy-to-gh-pages.yml/badge.svg)](https://github.com/Damagevindi/hm_elon_morsk/actions/workflows/deploy-to-gh-pages.yml)

**🌐 Web-Version:** [https://damagevindi.github.io/hm_elon_morsk/](https://damagevindi.github.io/hm_elon_morsk/)

## Features

- Echte Sinuswellen-Töne (Punkte & Striche hörbar)
- Animierte Oszilloskop-Wellenform während der Wiedergabe
- Ein-Klick Abbrechen während der Wiedergabe
- Sauberes, modernes Design

## Lokale Entwicklung

```bash
flutter pub get
flutter run
```

## Deployment

Die Web-Version wird automatisch über GitHub Actions auf den `gh-pages` Branch deployed.
